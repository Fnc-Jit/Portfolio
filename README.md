# Jitraj Esh – Developer Portfolio

A modern, dark-themed developer portfolio built with vanilla HTML, CSS, and JavaScript.

🔗 **Live:** (portfolio-two-neon-22.vercel.app)

![Portfolio Preview](https://img.shields.io/badge/Status-Live-brightgreen)

## ✨ Features

- **Dark modern theme** with neon blue & purple accents
- **Floating tile grid** with hover animations
- **Particle background** – interactive canvas-based particle system
- **Glassmorphism cards** with backdrop blur effects
- **Modal overlays** for each section with smooth transitions
- **GitHub API integration** – auto-fetches latest repositories
- **Fully responsive** – mobile, tablet, and desktop
- **Vercel-ready** – zero-config static deployment

## 📂 Sections

| Tile | Description |

## 🛠 Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- Google Fonts (Inter)
- Canvas API (particle system)
- GitHub REST API

## 🚀 Deploy

Push to GitHub and connect to [Vercel](https://vercel.com) — auto-detects as a static site.

Or run locally:

```bash
python3 -m http.server 8080
```
